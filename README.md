# py-picToWork

一个 Python 项目

## 快速开始

### 运行程序

```bash
python main.py
```

或者：

```bash
python3 main.py
```

### 安装依赖

```bash
pip install -r requirements.txt
```

## 项目结构

```
py-picToWork/
├── main.py           # 主程序入口
├── requirements.txt  # 项目依赖
└── README.md        # 项目说明
```

## 开发环境

- Python 3.7+
